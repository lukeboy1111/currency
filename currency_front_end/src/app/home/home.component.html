Currency Application

This app has three main areas associated with it:

1. Currency rates source

Currency rates can be pulled from European Central Bank website:

  • Reference Page
  • Current Rates Source
  • Historical Rates Source (last 90 days)

2. Conversion between currencies

Client UI allows selection of “Source” currency and “Destination” currency, entering the source amount and instant conversion into destination amount.

Example:

  • Source: USD, Amount: 20,000
  • Destination: EURO, Calculated Amount: [SOURCE_AMOUNT] * [FX_RATE]

Client UI displays the source of the rates (EU Cent. Bank or other) and the latest update date and time.

3. Conversion History

Client UI allows saving and displaying conversion results (see para 2 “Conversion between currencies”).

Users can save current conversion state with these parameters:

  • Source Currency, Source Amount
  • Destination Currency, Calculated Amount
  • Date/Time, Rate Source and Notes.
